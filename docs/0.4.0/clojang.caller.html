clojang.caller

call

(call func)(call func args)

This function is intended to be used under the hood by all Clojang functions that need to call jiface functions, catching all exceptions (should they arise), returning either the result of the jiface call or error data in the event of an underlying exception.

call!

(call! func)(call! func args)

This function is intended to be used under the hood by all Clojang functions that need to call jiface functions but that donâ€™t return results. The intent is to normalize such calls (which in Erlang sometimes return true or ok) with a consistent side-effects result: :ok. Since this function calls to call above, the same applies for any underlying exceptions that arise: they are converted to data and returned as a data structure.
